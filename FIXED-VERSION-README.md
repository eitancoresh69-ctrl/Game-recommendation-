## 🚀 שלוש צעדים להתחלה

### 1️⃣ התקן
```bash
pip install -r requirements.txt
```

### 2️⃣ הגדר API keys
```bash
mkdir -p .streamlit
```

יצור ``.streamlit/secrets.toml`:
```toml
GEMINI_API_KEY = "your-gemini-key"
```

קבל חינם מ: https://makersuite.google.com/app/apikey

### 3️⃣ הפעל
```bash
streamlit run index.html
```

פתח: `http://localhost:8501`

## 🔧 בעיות שכיחות

### ❌ "No games found"
**פתרון:** 
- בדוק שהתאריך הנוכחי לא עבר בניגוד ל-API
- בדוק חיבור internet

### ❌ "משחקים שהתחילו עדיין מוצגים"
**פתרון:**
- ודא שיש לך את `api_sofascore.py` החדש
- בדוק שהשעה של השרת היא בערך נכון

### ❌ "Timezone errors"
**פתרון:**
- הקוד גם לך טופל את DST (שעון קיץ) אוטומטי
- לא צריך לעשות שום דבר
